mpegCoder

This is a C++ based FFmpeg Encoder/Decoder for Python 3.5+ & numpy 1.13+. Both Linux & Win versions are provided. Theoretically you do not need to install FFmpeg for using this library.

Getting startedPyPI Project

Python without depdencies

Implemented by the Python-C-API. This feature has been included in stdlib. No matter when you need to compile or use this package, you do not need any python dependencies.

Fully based on FFMpeg

Built with the shared-lib-enabled FFMpeg. Since the original FFMpeg is not modified. All encoders and decoders are available. We have also provided a packed mpegCoder with all dynamic libraries.

Compiled by C++

Compiled by C++ on both Windows and Linux. The Win version and the Linux version are compiled by VC++ and G++ respectively. All source codes of this project are open-sourced by GPL v3 License.
