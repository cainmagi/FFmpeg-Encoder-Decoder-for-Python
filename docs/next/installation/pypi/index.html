Installation from PyPI

To install the pre-compiled package, just run

pip install mpegCoder

The PyPI repository is supported since mpegCoder 3.1.0. We support from python 3.5 to python 3.9. To check the details of each pre-compiled version, please view the manual installation guides for Windows and Linux.

The package installed by this method is shipped with all required dynamic libraries. Users do not need to install any other dependencies in this case. However, if users find that the package could not be imported after the installation, please check the troubleshooting page first.
